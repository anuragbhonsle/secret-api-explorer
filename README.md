A simple Express.js app that explores different authentication methods using secrets API(https://secrets-api.appbrewery.com/).

Features

- No Auth request
- Basic Auth (username + password)
- API Key Auth
- Bearer Token Auth
- Styled UI with EJS + CSS (glassmorphism, modern layout)

Tech Stack

- Node.js
- Express.js
- Axios
- EJS
- dotenv
